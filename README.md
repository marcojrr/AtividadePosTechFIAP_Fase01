# Atividade PósTech FIAP - Fase 01

## Objetivo do projeto

Estamos lidando com uma empresa em crescimento, que vem enfrentando dificuldades por cota do aumento no volume de clientes, pedidos e entregas com os quais precisa lidar. O principal ponto de atenção é a variabilidade do NPS (Net Promoter Score) entre os diferentes perfis de clientes atualmente atendidos, uma vez que se observou que alguns se tornam promotores da marca, enquanto outros se tornam detratores. O problema de negócio consiste em compreender quais ações estão gerando reações positivas e, de forma ainda mais relevante, identificar aquelas que estão provocando reações negativas.
